## 2. Description of Service

chat2repo is a free browser extension that allows you to:
- Capture ChatGPT conversations and save them to GitHub
- Capture selected text from any web page and save to GitHub
- Organize captured content with tags and auto-generated file paths
- Create markdown notes with YAML front-matter
